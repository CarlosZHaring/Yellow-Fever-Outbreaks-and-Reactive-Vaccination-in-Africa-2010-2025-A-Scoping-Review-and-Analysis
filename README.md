# Yellow-Fever-Outbreaks-and-Reactive-Vaccination-in-Africa-2010-2025-A-Scoping-Review-and-Analysis
This is a repository containing the data (excel and csv format) and the R code for the anaylsis seen in the final paper. The YF_outbreaks csv and excel files contains information on the outbreaks analyzed and the YF_RVCs csv and excel files contain information on the yellow fever reactive vaccinations analyzed. The excel files has color-coded cells that match with the corrisponding links to the sources used to find the information for that cell, comments on cells are provided for additional clarification as needed. Data dictionaries are provided for all files.
